ENDPOINTS:

  • USER
    • POST /login: return token
  • ROOMS (required LOGIN)
    • GET /rooms: return all rooms
    • POST /rooms: create a new room
    • GET /rooms/:id : receive a booking
    • PATCH /rooms/:id - update a room
  • BOOKINGS (required LOGIN)
    • GET /bookings: return all bookings
    • POST /bookings: create a new booking
    • GET /booking/:id - receive a booking
    • PATCH /booking/:id - update a booking
  • REVIEWS (required LOGIN)
    • GET /reviews: return all review
    • POST /reviews: create a new review
    • GET /reviews/:id - receive a review
    • PATCH /reviews/:id - update a review
